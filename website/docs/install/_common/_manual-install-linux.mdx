### Download

The GNU/Linux versions of **xPack flex**
are packed as `.tar.gz` archives.
Download the latest version named like:

* <code>xpack-flex-{ customField('upstreamVersion') }-{ customField('xpackSubversion') }-linux-x64.tar.gz</code>
* <code>xpack-flex-{ customField('upstreamVersion') }-{ customField('xpackSubversion') }-linux-arm.tar.gz</code>
* <code>xpack-flex-{ customField('upstreamVersion') }-{ customField('xpackSubversion') }-linux-arm64.tar.gz</code>

:::note

In case you wonder where the suffix comes
from, it is exactly the Node.js `process.platform` and `process.arch`.

:::

### Unpack

To manually install the <b>xPack flex</b>,
unpack the archive and move it to a location of your choice.

The recommended location is
the <code>~/.local/xPacks/flex</code> folder, for
example <code>/home/ilg/.local/xPacks/flex/xpack-flex-{ customField('upstreamVersion') }-{ customField('xpackSubversion') }</code>:

<CodeBlock language="console"> {
`mkdir -p ~/.local/xPacks/flex
cd ~/.local/xPacks/flex

tar xvf ~/Downloads/xpack-flex-${ customField('upstreamVersion') }-${ customField('xpackSubversion') }-linux-x64.tar.gz
chmod -R -w xpack-flex-${ customField('upstreamVersion') }-${ customField('xpackSubversion') }`
} </CodeBlock>
