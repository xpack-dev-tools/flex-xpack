The main target of this project is macOS 10.13, as the older version of
this utility fails during some builds.

[flex](https://github.com/westes/flex) is a tool for generating scanners: programs which recognize lexical patterns in text.

This is a fork of the original [flex](https://sourceforge.net/projects/flex/), maintained up to 2.6.0 (2015).

**flex** is an open source project hosted on
[GitHub](https://github.com/westes/flex)
and maintained by Will Estes.
